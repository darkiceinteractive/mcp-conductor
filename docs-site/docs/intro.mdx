MCP Conductor is a proxy that sits between Claude and your MCP servers, compressing tool responses by up to 99.7% and adding code execution capabilities.

## What is MCP Conductor?

- **Token savings**: 88–99% compression on typical MCP tool responses
- **Code execution**: Run JavaScript, Python, TypeScript, and more inside MCP calls
- **Drop-in proxy**: No changes required to your existing MCP servers
- **Observable**: Built-in metrics, progress reporting, and cancellation
